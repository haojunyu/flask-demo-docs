本文档使用 vuepress 进行静态页面生成，详见 vuepress 文档

本项目中的 md 文件并不完全遵从 Markdown 语法，其中使用了部分 vuepress 支持的高亮语法，详见这里


## 启动方式

```bash
# 安装依赖
npm install
# debug
npm install && npm run dev
# build
npm install && npm run build
```
